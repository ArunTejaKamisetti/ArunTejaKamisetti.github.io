How file uploads work on a static site:

  1. Put your file (PPT exported as PDF, résumé, image) into the project's assets/ folder — you can create subfolders like assets/decks/.
  2. Reference it here: for a résumé use the Résumé file path field above; for a deck, paste its path/URL in the deck's link.
  3. Save & sync, then run deploy.bat (or commit & push). The file ships with your repo and shows on the live site.

Tip: for Google Slides decks, use File → Share → Publish to web → Embed and paste that URL in the deck's Embed URL — no file upload needed.

After saving, replace content.json + index.html in your repo, then push.
